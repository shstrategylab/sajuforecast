# 사주 예측 프롬프트 생성기

생년월일시를 입력하면 **Claude AI 전용 정밀 사주 분석 프롬프트**를 자동으로 생성합니다.

## 기능

- 생년월일시 + 성별 입력
- 사주팔자 자동 산출 (연주·월주·일주·시주)
- 오행 강약·십성·충형합 자동 계산 및 프롬프트 반영
- 대운·세운 자동 배열
- 인생 대소사 15개 도메인 선택 분석
  - 결혼·동거·재혼
  - 이혼·별거·이별
  - 연애·이성운
  - 출산·자녀운
  - 취업·승진·이직
  - 사업성패·창업
  - 재물획득·손실·투기
  - 건강이상·수술·질병
  - 사고·부상·재난
  - 소송·관재·법적문제
  - 이사·이민·해외운
  - 학업·시험·자격증
  - 명예·성공·몰락
  - 부모사망·형제갈등·가족운
  - 사망·생사 관련 극한운

## 사용법

1. `index.html`을 브라우저로 열거나 GitHub Pages로 배포
2. 생년월일시 입력
3. 분석받을 영역 선택
4. **프롬프트 생성하기** 클릭
5. **복사하기** → Claude.ai 대화창에 붙여넣기

```
    ├── saju.js         # 사주 계산 엔진 (천간·지지·십성·충형합·대운·세운)

    └── main.js         # UI 이벤트 핸들러
```

### 산출 우선순위 (정밀도 순)

| 기둥 | 1순위 (정밀) | 2순위 (근사 폴백) |
|---|---|---|
| 연주 | 절기 DB로 입춘 날짜 확정 + 만세력 DB `year_ganji` 조회 | 60갑자 순환 공식 |
| 월주 | 절기 DB로 정확한 절기 경계 탐색 + 오호둔법(五虎遁法) | 월 숫자 기준 근사 |
| 일주 | 만세력 DB `daily_cache` 직접 조회 (1950~2027) | `jan_1st_ganji` + 경과일수 환산 (~2050) |
| 대운 시작 나이 | 절기까지 실제 일수 ÷ 3 (전통 공식) | 월중 날짜 기준 근사 (3~9세) |

- 절기 DB 범위(2010~2050)를 벗어나는 생년은 월주·대운이 **근사치**로 계산되며, 생성된 프롬프트 상단에 `⚠` 표시로 자동 안내됩니다.
- 절기 DB의 `ganji_year` 텍스트 필드는 연도별 표기 기준이 일관되지 않아 신뢰도가 낮아 사용하지 않으며, 입춘 **날짜**만 경계값으로 사용하고 실제 간지는 만세력 DB로 산출합니다.

## 분석 구조 (프롬프트 내 포함 항목)

### FOUNDATION LAYER (원국 분석)
- 오행 강약 분포
- 일간 강약 (신강/신약)
- 용신·희신·기신·구신·한신 확정
- 십성 배치 및 육친 분석
- 격국 확정
- 형·충·합·파·해·공망·역마·도화 분석

### TIMELINE LAYER (운세 흐름)
- 대운 (10년 주기)
- 세운 (1년)
- 월운 (월 단위 정밀화)

### 용신 발동 사이클
- 활성 / 약화 / 무력화 / 재발동 구간 추적

### EVENT DOMAIN (인생 대소사)
- 도메인별 십성·충형합·용신 발동 기준 완전 적용

## 주의사항

- 절기 DB는 2010~2050년만 커버합니다. 이 범위를 벗어나는 생년월일은 월주·대운이 근사치로 계산되며, 생성된 프롬프트에 자동으로 ⚠ 표시됩니다.
- 만세력 DB는 1950~2050년을 커버합니다. 이 범위를 벗어나면 60갑자 순환 공식으로 폴백합니다.
- 생성된 프롬프트는 Claude.ai (claude.ai)에서 사용하도록 설계되었습니다.
