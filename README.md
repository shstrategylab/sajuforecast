# 사주 예측 프롬프트 생성기

생년월일시를 입력하면 **Claude AI 전용 정밀 사주 분석 프롬프트**를 자동으로 생성합니다.

## 기능

- 생년월일시 + 성별 입력
- 사주팔자 자동 산출 (연주·월주·일주·시주)
- 오행 강약·십성·충형합 자동 계산 및 프롬프트 반영
- 대운·세운 자동 배열
- 인생 대소사 15개 도메인 선택 분석
  - 결혼·동거·재혼
  - 이혼·별거·이별
  - 연애·이성운
  - 출산·자녀운
  - 취업·승진·이직
  - 사업성패·창업
  - 재물획득·손실·투기
  - 건강이상·수술·질병
  - 사고·부상·재난
  - 소송·관재·법적문제
  - 이사·이민·해외운
  - 학업·시험·자격증
  - 명예·성공·몰락
  - 부모사망·형제갈등·가족운
  - 사망·생사 관련 극한운

## 사용법

1. `index.html`을 브라우저로 열거나 GitHub Pages로 배포
2. 생년월일시 입력
3. 분석받을 영역 선택
4. **프롬프트 생성하기** 클릭
5. **복사하기** → Claude.ai 대화창에 붙여넣기


## 분석 구조 (프롬프트 내 포함 항목)

### FOUNDATION LAYER (원국 분석)
- 오행 강약 분포
- 일간 강약 (신강/신약)
- 용신·희신·기신·구신·한신 확정
- 십성 배치 및 육친 분석
- 격국 확정
- 형·충·합·파·해·공망·역마·도화 분석

### TIMELINE LAYER (운세 흐름)
- 대운 (10년 주기)
- 세운 (1년)
- 월운 (월 단위 정밀화)

### 용신 발동 사이클
- 활성 / 약화 / 무력화 / 재발동 구간 추적

### EVENT DOMAIN (인생 대소사)
- 도메인별 십성·충형합·용신 발동 기준 완전 적용

## 주의사항

- 월주 산출은 절기(節氣) 기반 정밀 계산이 아닌 월 기준 근사치입니다.
- 정밀한 사주는 만세력 또는 절기 DB와 함께 사용하는 것을 권장합니다.
- 생성된 프롬프트는 Claude.ai (claude.ai)에서 사용하도록 설계되었습니다.
